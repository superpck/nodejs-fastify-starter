# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog,
and this project follows Semantic Versioning.

## [1.0.0] - 2026-02-26

### Added
- Initial Node.js + Fastify starter project.
- TypeScript setup with ESM (`tsx`, `typescript`).
- JWT authentication with `@fastify/jwt`.
- Knex integration with MySQL (`mysql2`) and PostgreSQL (`pg`) support.
- Health and auth routes.
- User controller and user routes.
- Centralized protected route scope in `src/routes/index.ts`.
- Project documentation updates and GitHub Copilot credit.
